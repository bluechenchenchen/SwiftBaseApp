## 🚀 技术栈和依赖

### 开发技术

- **Swift 5.9+**
- **SwiftUI 5.0+**
- **iOS 16.0+**
- **Xcode 15.0+**

### 第三方依赖

- **Down** - Markdown 渲染
- **Charts** - 图表展示（性能分析）
- **SwiftUI-Introspect** - UIKit 集成（调试工具）

### 系统框架

- **Foundation** - 基础功能
- **UIKit** - 底层控件
- **Combine** - 响应式编程
- **CoreData** - 数据持久化
